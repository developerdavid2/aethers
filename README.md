## 🚀 Quick Start

### Prerequisites

- Node.js ≥ 18.0.0
- pnpm ≥ 9.0.0 (`npm install -g pnpm`)
- Xcode (for iOS)
- Android Studio (for Android)
- Convex CLI (`npm install -g convex`)

### 1. Install Dependencies

```bash
pnpm install
```
